# timeline-history-database
时序编录——本地历史时序数据库工具
纯前端离线工具，所有数据存储在浏览器 LocalStorage，不需要服务器。

## 核心功能
1. 多档案管理：新建、重命名、删除独立档案
2. 完整时间录入：起止年/月/日、年号备注
3. 人物分类：主线人物、非主线人物；三种事件重要标记
4. 双视图切换：表格视图 / 时间轴视图
5. 全局关键词搜索，支持上下跳转匹配条目
6. JSON 导入导出，一键备份/恢复全部档案
7. 滚动内容时顶部操作栏固定，完整展示结束年月日

## 使用方法
1. 下载仓库内 `index.html` 文件；
2. 双击文件，用任意浏览器打开即可使用；
3. 数据自动保存在当前浏览器，清除缓存会丢失数据，请定期导出JSON备份。
